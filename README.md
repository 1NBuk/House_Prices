# House Prices Prediction

## Цель проекта
Построить модель для прогнозирования цен на жильё на основе данных Kaggle (House Prices: Advanced Regression Techniques).

Исследовать данные, обработать пропуски и выбросы, подобрать оптимальные модели.

Сравнить разные модели (Linear Regression, Decision Tree, XGBoost, LightGBM, CatBoost) и создать ансамбль.

Продемонстрировать интерпретируемость модели с SHAP.

## Описание данных

Данные содержат 1459 строк train и 1458 строк test, 76 признаков (числовые и категориальные).

Целевая переменная: SalePrice.

Признаки включают: характеристики дома, участка, гаража, подвала и пр.

## Этапы работы
a) Анализ данных (EDA)

Проверка на пропуски, дубликаты и выбросы.

Визуализация распределения цен, выбросов и корреляций с целевой переменной (SalePrice).

Построение тепловых карт и scatter plot для топ-признаков.

b) Предобработка данных

Удаление колонок с очень большим количеством пропусков (PoolQC, Alley, MiscFeature, Fence).

Заполнение пропусков:

Числовые — медианой.

Категориальные — модой.

Масштабирование числовых признаков (StandardScaler) и кодирование категориальных (OneHotEncoder).

Замена выбросов на границы IQR.

c) Разделение на train/test
train_test_split с test_size=0.2.

## Модели

Линейная регрессия

Дерево решений + GridSearchCV

XGBoost

LightGBM

CatBoost

Ансамбль (Stacking): XGBoost + LightGBM + CatBoost


## Метрики оценки

MAE, MSE, R²

RMSLE для сравнения с Kaggle метрикой

Визуализация:

True vs Predicted

Residuals

## Интерпретация модели

SHAP для XGBoost: графики важности признаков и влияние каждого признака на прогноз.


## Результаты

Сравнение моделей по метрикам: R², MAE, MSE, RMSLE.

Ансамбль показал наилучший результат.

Выводы:

Наиболее важные признаки для цены: OverallQual, GrLivArea, GarageCars, TotalBsmtSF и др.

Буcтинг модели (XGBoost/LightGBM/CatBoost) значительно улучшают точность по сравнению с линейной регрессией.

Предлагаю остановиться на XGBoost, так как у него примерно похожие метрики с ансамблем моделей, однако он быстрее обучается
